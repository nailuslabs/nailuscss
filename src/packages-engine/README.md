# nailuscss Core Packages

The core packages for nailuscss engine, they should be aimed for agnostic use cases.

| Package                                | Description                       | Included in `nailuscss` |
| -------------------------------------- | --------------------------------- | -------------------- |
| [@nailuscss/core](./core)                 | The core library without preset   | ✅                   |
| [@nailuscss/cli](./cli)                   | Command line interface for nailuscss | ✅                   |
| [@nailuscss/autocomplete](./autocomplete) | Utils for autocomplete            | No                   |
| [@nailuscss/config](./config)             | Configuration file loader         | ✅                   |
